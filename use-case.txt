# Use-Case Диаграмма для лабораторной системы

```
+--------------------------+
|       Лабораторная       |
|        Система          |
+--------------------------+
           |
           | (включает)
           v
+--------------------------+
|   Основные Акторы:       |
|--------------------------|
| 1. Администратор        |
| 2. Лаборант             |
| 3. Бухгалтер            |
| 4. Пациент              |
+--------------------------+
           |
           | (взаимодействуют)
           v
+--------------------------+
|       Use-Cases:         |
|--------------------------|
| 1. Управление пользоват.|
| 2. Вход в систему       |
| 3. Забор крови          |
| 4. Проведение анализов  |
| 5. Просмотр результатов |
| 6. Управление услугами  |
| 7. Финансовый учет      |
| 8. Генерация отчетов    |
| 9. Управление анализатор|
+--------------------------+
```

## Детализация Use-Cases:

1. **Управление пользователями** (Администратор)
   - Создание/редактирование учетных записей
   - Назначение прав доступа

2. **Вход в систему** (Все роли)
   - Аутентификация
   - Ведение истории входов

3. **Забор крови** (Лаборант)
   - Регистрация образцов
   - Назначение штрих-кодов

4. **Проведение анализов** (Лаборант)
   - Ввод результатов
   - Использование анализаторов
   - Проверка качества

5. **Просмотр результатов** (Пациент, Лаборант)
   - Доступ к личным данным
   - Просмотр истории анализов

6. **Управление услугами** (Администратор)
   - Добавление/изменение тестов
   - Установка цен и сроков

7. **Финансовый учет** (Бухгалтер)
   - Учет выполненных анализов
   - Формирование счетов

8. **Генерация отчетов** (Администратор, Бухгалтер)
   - Статистика по анализам
   - Финансовая отчетность

9. **Управление анализаторами** (Администратор)
   - Регистрация оборудования
   - Настройка параметров

## Связи:
- Администратор имеет доступ ко всем функциям управления
- Лаборант работает с забором крови и проведением анализов
- Бухгалтер отвечает за финансовые аспекты
- Пациент может только просматривать свои результаты
